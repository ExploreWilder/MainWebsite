QMapShack

Plan, edit, and share tracks on interactive and immersive maps.

If you like the QMapShack desktop application, and the 3D map player available in my stories, then you would also like QMapShack by ExploreWilder combining both! Notice that my implementation of QMapShack is only available on a GNU/Linux-based operating system such as Ubuntu. Only the source code of the application is currently available, so you will have to compile the project by yourself. Good luck and have a great experience!
